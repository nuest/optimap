OGC API — Features

OPTIMAP publishes its data as a standards-compliant OGC API — Features endpoint. Any WFS3-capable client (QGIS, R sf, Python geopandas) can connect directly.

  • Landing page — service description and links
  • Browse works — published works as GeoJSON features

Supports bbox, datetime, limit, and offset query parameters. See API documentation for the full schema.
